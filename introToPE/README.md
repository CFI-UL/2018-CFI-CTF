## Writeup

Click on `IntroToPE`

Go to the `IntroToPE.exe` menu

Open the `IntroToPE` classes

Go to the `verifyPasswd` class

Analyze the `verifyPasswd` function

Decode the string using [https://kt.gy/tools.html](https://kt.gy/tools.html)
